Snake Game (Java Swing)
A classic implementation of the nostalgic Snake Game built using the Java Swing framework. This project demonstrates basic game loops, event handling (KeyListeners), and 2D graphics rendering.

🎮 Features
Classic Gameplay: Navigate the snake to eat apples and grow longer.

Score Tracking: Real-time score updates as you play.

Game Over State: Detects collisions with walls or the snake's own body.

Responsive Controls: Smooth movement using the arrow keys.

🛠️ Tech Stack
Language: Java

GUI Library: Swing / AWT

Tools: Java Development Kit (JDK) 8 or higher
